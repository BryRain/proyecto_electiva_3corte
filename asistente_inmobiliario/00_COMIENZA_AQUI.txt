💾 INSTALACIÓN RÁPIDA
═════════════════════════════════════════════════════════════════════════════════

Windows:
  1. python -m venv venv
  2. venv\Scripts\activate
  3. pip install -r backend\requirements.txt
  4. copy .env.example .env (editar con API keys)
  5. python backend\database\init_db.py
  6. python backend\app.py
  7. Abrir: http://localhost:5000

Linux/Mac:
  1. python3 -m venv venv
  2. source venv/bin/activate
  3. pip install -r backend/requirements.txt
  4. cp .env.example .env (editar con API keys)
  5. python backend/database/init_db.py
  6. python backend/app.py
  7. Abrir: http://localhost:5000
